# The Seeds - Life Simulator (v0.6.2)

The Seeds คือโปรเจกต์เกมแนว Sandbox Life Simulator และ Ecosystem Simulation บนเว็บเบราว์เซอร์ พัฒนาโดย Studio Balmung และ Neofilisoft ผู้เล่นจะได้รับบทบาทเป็นผู้ควบคุมระบบนิเวศน์เพื่อสร้าง แทรกแซง และสังเกตการณ์การวิวัฒนาการของสิ่งมีชีวิตรวมถึงการก่อตั้งเผ่าพันธุ์ต่างๆ

## ฟีเจอร์หลักของระบบ

* **ระบบโลกและสภาพอากาศแปรปรวน (Dynamic World and Seasons):** แผนที่ขนาด 100 x 100 บล็อก ประกอบด้วยสภาพแวดล้อมที่หลากหลาย เช่น Plain, Forest, Desert, Wetland, Tundra, Mountain, River, Ocean โดยมีระบบจำลองฤดูกาล (Spring, Summer, Autumn, Winter) ที่ส่งผลกระทบต่ออุณหภูมิและความชื้นในแต่ละพื้นที่โดยตรง
* **ระบบพันธุกรรมขั้นสูง (Deep Genetics and Evolution):** สิ่งมีชีวิตทุกตัวมีโครโมโซมแบบคู่ (Diploid) ควบคุมลักษณะทางกายภาพ พฤติกรรม และความสามารถ เช่น พลังโจมตี ปริมาณการบริโภค พลังงาน ความเร็ว และอัตราการกลายพันธุ์ (Mutation Rate)
* **ระบบเผ่าพันธุ์และการทูต (Tribe and Politics):** เมื่อสิ่งมีชีวิตมีระดับสติปัญญา (Intelligence) สูงพอ จะสามารถรวมกลุ่มกันตั้งเผ่าพันธุ์ มีการสร้างธงประจำเผ่า สะสมทรัพยากร ขยายอาณาเขต และมีระบบความสัมพันธ์ระหว่างเผ่า (พันธมิตร, เป็นกลาง, สงคราม) รวมถึงการเลือกผู้นำสูงสุด (King)
* **ระบบพลังอำนาจควบคุม (God Powers):** ผู้เล่นสามารถแทรกแซงระบบนิเวศน์ได้ผ่านคำสั่งพิเศษ:
    * Spawn Life: การสร้างสิ่งมีชีวิตใหม่ลงในพิกัดที่กำหนด
    * Nuke: การระเบิดทำลายล้างพื้นที่และสิ่งมีชีวิตในบริเวณนั้น
    * Cull Weak: การกำจัดสิ่งมีชีวิตที่มีค่าความเหมาะสมทางพันธุกรรม (Fitness) ต่ำกว่าเกณฑ์ออกไปจากระบบ
    * Anoint King: การสถาปนาสิ่งมีชีวิตที่แข็งแกร่งที่สุดให้เป็นผู้นำเผ่า
    * God War: การจุดชนวนสงครามระหว่างเผ่าพันธุ์
* **ระบบบันทึกข้อมูล (Saves and Autosave):** รองรับระบบบันทึกเกมผ่าน LocalStorage และมีระบบบันทึกอัตโนมัติ (Autosave) ทุกๆ 1 นาที

## โครงสร้างยีนและพันธุกรรม

สิ่งมีชีวิตในระบบขับเคลื่อนด้วยยีนที่ส่งต่อผ่านการสืบพันธุ์และการสุ่มกลายพันธุ์:

* ลักษณะทางกายภาพ: size (ขนาดตัว), speed (ความเร็ว), atk (พลังโจมตี), def (พลังป้องกัน), lifespan (อายุขัย)
* พฤติกรรมและระบบเผาผลาญ: metab (อัตราการเผาผลาญพลังงาน), reproThreshold (ระดับพลังงานที่พร้อมสืบพันธุ์), aggro (ความดุร้าย), intel (ระดับสติปัญญา)
* การปรับตัวและความทนทาน: mutRate (โอกาสเกิดการกลายพันธุ์), tempPref (อุณหภูมิที่เหมาะสม), humidPref (ความชื้นที่เหมาะสม)

## โครงสร้างไฟล์ในโปรเจกต์

โปรเจกต์นี้พัฒนาขึ้นด้วย Vanilla JavaScript โดยไม่มีการใช้ Dependencies ภายนอก:

* index.html: โครงสร้างหน้าจอหลัก หน้าจอเริ่มเกม แผงควบคุมระบบ และเครื่องมือตรวจสอบพันธุกรรม (Genetics Inspector)
* style.css: การจัดการสไตล์และการแสดงผลอินเตอร์เฟสในรูปแบบ Retro Sci-fi โดยใช้ JetBrains Mono เป็นฟอนต์หลัก
* script.js: กลไกหลักของระบบ (Game Loop, Pathfinding, Genetics, Physics, Rendering)

## วิธีการใช้งานและเริ่มต้นระบบ

1. ดาวน์โหลดหรือคัดลอกไฟล์ทั้งหมดในโปรเจกต์ไปไว้ในโฟลเดอร์เดียวกัน
2. เปิดไฟล์ index.html ด้วยเว็บเบราว์เซอร์ (เช่น Chrome, Firefox, Edge, Safari)
3. เลือกปุ่ม New World เพื่อสุ่มสร้างระบบนิเวศน์ใหม่ (หรือระบุค่า Seed ที่ต้องการในช่องรับข้อมูล)
4. ระบบจะเริ่มจำลองและดำเนินไปโดยอัตโนมัติ ผู้เล่นสามารถใช้แผงควบคุมด้านขวาเพื่อแทรกแซงระบบได้ทันที

## การควบคุมและสั่งการ

* คลิกเมาส์ซ้ายค้างแล้วลาก (Click and Drag): สำหรับการเลื่อนมุมกล้อง (Pan Camera)
* ลูกกลิ้งเมาส์ (Mouse Wheel): สำหรับการซูมเข้าและซูมออก (Zoom In / Out)
* คลิกที่ตัวสิ่งมีชีวิต: เพื่อเปิดเครื่องมือ Genetics Inspector สำหรับตรวจสอบข้อมูลยีนและการแสดงผลค่าพลังโดยละเอียด
* แผงควบคุมด้านขวา: สำหรับเลือกใช้งานพลังจำลอง ดูประวัติเหตุการณ์ (Log) และสถิติภาพรวมของโลก
